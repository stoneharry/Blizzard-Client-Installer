기본 설정:  설정 메뉴를 살펴보려면 Esc 키를 누르면 됩니다. 기본 설정 화면에선 비디오 설정이나 키 설정 등의 게임 설정을 변경할 수 있습니다.

참고:  전체 키 목록을 살펴보려면 Esc 키를 누르고 '단축키 설정'을 선택하시기 바랍니다. 현재 설정 가능한 모든 행동의 단축키를 보실 수 있습니다.

캐릭터 이동:  W,A,S,D 키를 사용하여 캐릭터를 이동할 수 있습니다. 또는 마우스의 양쪽 버튼을 동시에 누르면 앞으로 이동할 수 있습니다(이 때 마우스를 움직이면 시점을 변환할 수 있습니다). 만약 게임 설정에서 "마우스 이동" 옵션을 켜놓으셨다면 이동하고자 하는 지점을 마우스 오른쪽 버튼으로 클릭하여 캐릭터를 이동할 수 있습니다.

시점 변경:  왼쪽 마우스 버튼을 누르고 마우스를 움직이면 주위를 둘러볼 수 있습니다. 오른쪽 마우스 버튼을 누르고 움직이면 캐릭터의 방향을 바꿀 수 있습니다. 마우스 가운데 버튼이나 키보드의 Home 키와 End 키를 사용하여 시점을 확대하거나 축소할 수 있습니다.

행동하기:  마우스 왼쪽 버튼은 물체나 NPC를 선택하는 데 사용됩니다. 마우스 커서는 현재 대상과 가능한 행동들을 나타냅니다. 상호 작용이 가능한 대상에 마우스 커서를 가져가면 커서 모양이 변하며 이 대상에 오른쪽 버튼을 누르면 해당 행동을 할 수 있습니다(검 모양 = 공격, 톱니 모양 = 작동, 말풍선 = 대화 등).

공격하기:  적을 공격하려면 사정 거리로 접근하여(공격 가능한 대상에 마우스를 가져갔을 때 검 모양의 커서가 회색으로 나타난다면 사정 거리를 벗어난 것입니다) 마우스 오른쪽 버튼을 클릭 해야 합니다. 공격 모드로 전환되었을 때 적과 거리가 충분하다면 자동으로 공격을 시작할 것입니다. 마우스를 클릭하는 대신에 행동 단축바에 있는 '공격' 아이콘을 클릭해도 됩니다.

단축 아이콘바:  이것은 화면 하단에 있는 각종 주문이나 특수 능력, 음식물(생명력과 마나 회복에 사용)과 같이 자주 사용되는 아이콘을 등록시킬 수 있는 단축 아이콘바를 말합니다. 등록된 아이콘을 실행하려면 해당 아이콘에 마우스로 왼쪽 클릭하거나 해당 키보드의 버튼을 누르면 됩니다. 등록 된 주문이나 특수 능력 아이콘의 좀 더 자세한 설명을 참고하려면 하단 메뉴의 마법책과 특수 능력 아이콘을 클릭하여 원하는 아이콘의 능력을 살펴볼 수 있습니다. 단축 아이콘바에는 자신이 원하는 주문이나 능력 아이콘을 새로 등록하거나 마우스 왼쪽 버튼으로 드래그하여 서로의 위치를 변경할 수 있습니다. 또한 소지품에 있는 치유 물약 등을 등록하여 사용할 수도 있습니다.

참고:  Shift 키를 누른 채로 마우스 가운데 버튼을 돌리면 해당 단축 아이콘바의 페이지를 빠르게 전환할 수 있습니다. Shift 키와 키보드 화살표 키를 동시에 눌러도 같은 기능을 수행합니다.

파티, 거래, 다른 플레이어와의 상호 작용:  누군가를 파티에 초대하려면 마우스 왼쪽 버튼을 클릭하여 상대방을 선택한 후 상대 플레이어의 초상화에 오른쪽 버튼을 클릭하면 가능한 메뉴가 표시됩니다. 초대하기는 상대방을 자신의 파티에 초대하는 기능이며 다른 여러 메뉴를 통해서 상대방의 장비를 살펴보거나 결투를 신청하는 등의 행동을 할 수 있습니다.

참고: 이 기능은 적대 연합의 플레이어에게는 사용할 수 없습니다.

소리 설정:  Ctrl+S를 누르면 효과음을 켜고 끌 수 있습니다. Ctrl+M을 누르면 배경음악을 켜고 끌 수 있습니다.

캐릭터 정보:  화면 하단의 캐릭터 초상화 아이콘을 클릭하면 캐릭터 정보 화면이 나타납니다. 여기서는 자신의 특성이나 평판, 숙련 기술 등을 살펴볼 수 있습니다.

레벨 올리기:  10 레벨이 되면 첫 특성 포인트를 받습니다. 인터페이스 창에서 특성 버튼을 클릭하여 특성 메뉴를 열어 현재 소지한 특성 포인트로 새로운 특성을 배울 수 있습니다. 각 직업에는 3개의 특성 분류가 있으며 해당 특성 아이콘에 마우스 커서를 가져가면 필요한 조건과 특성 포인트가 표시됩니다.

새로운 주문과 특수 능력:  주문이나 특수 능력은 자동으로 생기지 않습니다. 새로운 주문이나 특수 능력을 배우려면 마을에 있는 자신의 직업 상급자를 찾아가야 합니다. 또한, 직업 전용 퀘스트를 수행하여 기술을 배우는 등의 다른 방법도 있습니다.

전리품 획득:  몬스터의 시체에 커서를 놓고 클릭하면 전리품 화면이 나타납니다. 아이템에 다시 클릭하면 해당 아이템이 소지품으로 들어오게 됩니다. 소지품을 열어 보려면 화면 오른쪽 하단에 있는 가방에 커서를 놓고 마우스 버튼을 클릭하면 됩니다.

가방 추가하기:  다른 가방을 얻으면 자신의 가방 슬롯에 새로운 가방을 장착하여 아이템의 저장 공간을 늘릴 수 있습니다. 새로 추가된 가방은 소지품 가방과 동일하게 작용합니다.

슬래시 명령어:  아래의 명령어는 게임에서 대화창에 입력하여 사용할 수 있는 '슬래시(/) 명령'입니다. 엔터를 누르면 대화창을 불러올 수 있습니다.

참고:  Tab 키를 사용하여 가능한 명령어를 살펴볼 수도 있습니다. 예를 들어, 대화창에 '/a'만 입력한 후에 Tab 키를 누르면 'a'로 시작되는 모든 명령을 차례대로 살펴 볼 수 있습니다.
/채팅도움말, /채팅도움 - 모든 채팅 명령어 표시
/따라, /f- 대상을 선택하고 이 명령어를 사용하면 자동으로 해당 플레이어를 따라갑니다.
/party <할말> ('/p <할말>') - 파티원에게 메시지 보내기(현재 파티원만 볼 수 있습니다).
/r - 귓속말에 답장하기
/say <할말> ('/s <할말>') - 자신의 캐릭터 근처에 있는 플레이어에게 말하기
/누구, /who - 접속한 플레이어 목록보기
/외침 <할말> ('/y <할말>') - 자신의 캐릭터에서 멀리 떨어진 곳까지 외치기
/em <할말> - 감정 표현하기 (예를 들어, 캐릭터 이름이 "티란데"일 때 "/em 노래합니다." 라고 입력하면 '티란데가 노래합니다.'로 표시됩니다)
/플레이시간, /played - 게임을 플레이한 총 시간을 표시합니다.
/매크로, /macro - 매크로 설정창을 엽니다.
/cast <주문 이름> - 주문 이름을 입력하여 시전할 수 있습니다. ("/cast 화염구 (1 레벨)"). 이 명령어를 매크로에서 사용하려면 매크로 창과 마법책 창을 동시에 연 뒤에 Shift 키를 누르고 해당 마법 아이콘을 클릭하면 됩니다.

채팅 명령어: 

/#, /c, /csay - #번 채팅 채널에 메시지 보내기 (예: /1 안녕하세요)
/알림 <채널번호> - 플레이어의 채널 입장/퇴장 메시지 보이기/숨기기
/자리비움, /afk. /다른용무중, /dnd - 자리 비움 또는 다른 용무 중으로 설정
/영구추방, /ban, /추방해제, /unban - 채널에서 플레이어 영구추방 또는 해제
/채널정보, /채널목록, /chatlist <채널번호> - 채널 목록 또는 채널 참가자 목록 보기
/채널초대. /cinvite - 채널에 해당 플레이어 초대
/입장 <채널명> <비밀번호> - 해당 채널을 생성하거나 존재하는 채널에 입장합니다.
/추방, /kick - 채널에서 해당 플레이어 추방
/나가기, /퇴장, /leave <채널번호> - 해당 채널 또는 모든 채널에서 나가기
/관리자지정, /mod, /관리자박탈, /unmod - 플레이어의 관리 권한 변경
/관리, /moderate - 채널의 관리 기능 켜기/끄기
/대화차단, /mute, /대화허용, /unmute - 플레이어의 대화 권한 변경
/비밀번호, /암호, /pass <채널> <비밀번호> - 비밀번호 변경

길드 명령어:  일부 길드 명령은 길드장만 사용할 수 있습니다.

/길드정보, /ginfo - 길드에 대한 기본적인 정보
/g <할말> - 모든 길드원에게 메시지 보내기
/o <할말> - 모든 길드 관리자에게 메시지 보내기
/길드초대, /ginvite <플레이어> - 길드에 해당 플레이어 초대하기
/길드추방, /gremove <플레이어> - 길드에서 해당 플레이어 추방하기
/길드승급, /gpromote <플레이어> - 해당 플레이어 길드 등급 올리기
/길드강등, /gdemote <플레이어> - 해당 플레이어 길드 등급 내리기
/길드공지, /gmotd <할말> - 오늘의 길드 메시지 정하기
/길드탈퇴, /gquit - 길드에서 탈퇴하기
/길드목록, /groster - 전체 길드원 목록 보기
/길드위임, /gleader <플레이어> - 다른 플레이어에게 길드장 위임하기 (길드장 전용)
/길드해체, /gdisband - 길드 해체하기 (길드장 전용)
