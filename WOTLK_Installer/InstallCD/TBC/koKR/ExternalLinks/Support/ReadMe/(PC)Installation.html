설치하기 (PC)

월드 오브 워크래프트: 리치 왕의 분노 베타 클라이언트가 설치되어 있다면, 일반 클라이언트를 설치하기 전에 제거해 주시기 바랍니다.

리치 왕의 분노를 설치하려면 설치 프로그램을 다운로드하시기 바랍니다. 전체 파일의 다운로드가 완료되면 "다운로드 완료" 버튼을 클릭한 후, 다운로드한 Installer.exe 프로그램을 실행하여 게임 파일 설치를 시작하시기 바랍니다. 설치 과정은 수분에서 수십분 정도 소요됩니다.

리치 왕의 분노의 설치 DVD를 가지고 계신다면 리치 왕의 분노 설치 DVD를 DVD-ROM 드라이브에 넣으십시오. DVD가 인식되면 DVD-ROM을 열어 리치 왕의 분노 설치 아이콘을 클릭한 후, 화면에 표시되는 안내에 따라 "리치 왕의 분노 설치"를 선택하십시오.

Windows XP나 2000 사용자: DVD를 DVD-ROM에 넣었을 때 자동으로 설치 메뉴가 나타나지 않는 경우에는 바탕 화면의 내 컴퓨터를 두 번 클릭하고, 리치 왕의 분노 DVD가 들어 있는 드라이브를 두 번 클릭한 다음 Setup 또는 Setup.exe라고 표시된 아이콘을 두 번 클릭하면 설치 메뉴가 화면에 표시됩니다.

Window Vista 사용자: DVD를 DVD-ROM에 넣었을 때 자동으로 설치 메뉴가 나타나지 않는 경우에는 시작 메뉴에서 "컴퓨터"를 클릭하고, 리치 왕의 분노 DVD가 들어 있는 드라이브를 클릭한 다음 설치 아이콘을 선택한 후 화면에 표시되는 안내에 따라 리치 왕의 분노를 설치하시기 바랍니다.

* 설치 전에 반드시 12GB 이상의 하드 디스크 여유 공간을 확보하시기 바랍니다.

참고: 리치 왕의 분노 설치 도중 문제가 발생하시면 설치 문제 해결란을 참고하시기 바랍니다.
