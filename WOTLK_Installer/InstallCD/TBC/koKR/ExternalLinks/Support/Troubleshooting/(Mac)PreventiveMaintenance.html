문제 예방 (Mac)

게임을 실행하기 전에 시스템을 최적화해 두면 많은 문제를 사전에 예방할 수 있습니다.

1. OS 업데이트: 월드 오브 워크래프트는 최신 업데이트가 적용된 운영체제에서 원활하게 실행됩니다. Mac 사용자께서는 Mac OS X가 업데이트되면 파일 권한 문제가 발생할 수 있습니다. 이 문제는 게임이 실행되지 않거나 게임 실행 도중 특정 작업을 수행하는 중에 예기치 않게 게임이 종료될 수 있습니다. 이러한 문제가 발생하면 다음 절차를 참고하셔서 월드 오브 워크래프트: 리치 왕의 분노가 설치된 하드 드라이브 내의 파일 권한 설정을 복구하시기 바랍니다.

Mac OS X 10.3, 10.4, 10.5 (Panther, Tiger, Leopard):
  • Dock 메뉴의 Finder 아이콘을 클릭합니다.
  • 응용 프로그램 폴더를 엽니다.
  • 유틸리티 폴더를 엽니다.
  • 디스크 유틸리티 응용 프로그램 아이콘을 더블 클릭합니다.
  • 게임이 설치된 볼륨을 선택합니다.
  • 검사/복구 항목을 선택합니다.
  • "디스크권한 복구"를 클릭합니다.

2. OS 설정: 운영체제에는 게임의 작동을 보다 원활하게 하고 오류를 줄이도록 변경할 수 있는 설정이 있습니다.

우선 전원 관리 기능을 끄거나 사용하지 않거나 화면 보호기 역시 게임 실행 전에 "없음"으로 설정하는 것이 좋습니다.

Mac OS X에서는 사용하지 않는 다른 응용 프로그램을 모두 종료하시기 바랍니다. Dock의 응용 프로그램 아이콘 아래 작은 삼각형의 유무로 실행 중인 응용 프로그램이 있는지를 확인하실 수 있습니다.

시스템 오버클러킹 중이면 시스템 내부의 온도가 너무 높아 프로그램의 성능에 영향을 미칠 수 있습니다. 오버클러킹 중인 아닌데도 시스템 내부의 열이 너무 높다면 시스템에 팬을 추가로 설치하여 케이스 내부의 열을 분산시키는 것이 좋습니다.

3. 드라이버 업데이트: 비디오 카드 및 사운드 카드 등의 하드웨어 드라이버는 항상 최신 버전을 사용하는 것이 중요합니다. 비디오 및 사운드 카드의 경우 하드웨어 제조회사의 웹 사이트에서 자신이 사용하는 모델에 해당하는 최신 드라이버를 확인하시기 바랍니다. 하드웨어 제조회사 목록은 블리자드 고객지원 페이지를 참조하십시오.

Apple 홈페이지에는 시스템에 장착된 ATI와 NVIDIA의 비디오 드라이버를 제공하고 있습니다. 새로운 드라이버는 Mac OS X 업데이트에 포함되어 있습니다. System Preferences 메뉴의 Software Update를 선택하시거나 애플컴퓨터코리아 Mac OS X 홈페이지를 방문하셔서 Mac OS X의 최신 업데이트가 있는지 확인하시기 바랍니다.

마우스에서 문제가 발생하는 일은 드문 편이지만 문제가 있다면 대부분 소프트웨어 업데이트를 통해 해결할 수 있습니다. 제조회사에서 제공하는 최신 장치 드라이버를 사용하십시오. 일부 포인팅 장치는 전용 소프트웨어를 사용하는데 이러한 소프트웨어도 함께 업데이트하는 것이 좋습니다. 문제가 지속되면 해당 소프트웨어를 종료한 다음 운영 체제에 포함된 범용 드라이버를 사용해 보십시오.

4. 하드 디스크 관리: Mac 컴퓨터에서는 설치된 Mac OS 버전과 호환되지 않는 "디스크 복구" 소프트웨어가 설치된 경우 예기치 않은 이유로 게임 파일이 손상될 수 있습니다. 게임을 재설치해야 한다면 게임을 설치하시기 전에 드라이브 관리 소프트웨어를 일시 중지하시기 바랍니다.
