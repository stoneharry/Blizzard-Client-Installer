Audio Problems (Mac)

Here are some of the most commonly experienced sound issues. If you are experiencing a problem that is not listed below, or the steps listed do not resolve the problem, please contact Technical Support.

1. Some sounds are playing much louder than others: This is often the result of inappropriate audio settings. You should delete the Config.wtf file so as to return your game to its original settings. This file can be found in the World of Warcraft WTF folder (/../WoW/WTF/Config.wtf).

2. Apple Macintosh crashes:  On Mac OS X, if you are experiencing frequent crashes, please make sure that you have QuickTime 7.5 installed. Run Software Update in System Preferences to upgrade your version of QuickTime to 7.5.
