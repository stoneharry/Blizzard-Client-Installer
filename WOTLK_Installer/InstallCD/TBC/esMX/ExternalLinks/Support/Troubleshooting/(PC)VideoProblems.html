Problemas de imagen (PC)

La mayoría de los problemas de imagen se deben a los programas y a sus configuraciones. Si experimentas un problema que no está listado abajo, o los pasos listados no resuelven el problema, contacta al departamento de soporte técnico.

1. La experiencia de juego es lenta o entrecortada:  la principal manera de mejorar el rendimiento del juego es a través de los ajustes de las propiedades gráficas dentro del juego. Cambiar a una configuración más baja puede aumentar la velocidad y la suavidad del juego a costa de calidad visual. Posiblemente quieras invertir algo de tiempo en experimentar con diferentes ajustes para encontrar la mejor calidad visual para una velocidad de juego aceptable.

Algunas propiedades se pueden ajustar utilizando el menú Opciones de Vídeo pulsando la tecla Esc mientras estás en el juego. Las propiedades también se pueden ajustar desde fuera del juego utilizando el archivo Config.wtf.

  • Abre el directorio de World of Warcraft: Wrath of the Lich King de tu disco duro.
  • Abre la carpeta WTF.
  • Abre el archivo Config.wtf usando un editor de texto, como Bloc de notas.

Estas propiedades están ideadas para permitir a usuarios avanzados la habilidad de personalizar el rendimiento del juego. Para rápidamente incrementar la velocidad de rendimiento, prueba algunas o todas de las siguientes instrucciones.

    SET lod "1"
    Cambia el nivel de detalle del terreno (LOD), encendido "1" y apagado "0". Habilitar esta característica permite al juego dibujar menos detalles cerca del horizonte, aumentando el rendimiento.
    SET lodDist "100"
    Ajusta la distancia a la que el nivel de detalle del terreno (LOD) ocurre. 100 es conservador, pero quizás quieras reducir este número a 50 para mejorar el rendimiento. LodDist solo funciona cuando LOD está a "1" pero puedes reducir el valor hasta 50 para aumentar el rendimiento.
    SET doodadAnim "0"
    Configurando esto a "1" activarás las animaciones ambientales, como los molinos de viento y los pájaros. Algunas áreas del juego tienen demasiados objetos animados, por lo que dejando esta opción a 0 puedes aumentar significativamente el rendimiento en dichas áreas.
    SET mapShadows "0"
    Desactiva la visualización de sombras del terreno ("1" las vuelve a activar). Esto mejorará la velocidad del juego sensiblemente.
    SET anisotropic "1"
    Puede ajustarse de 1 (off) a 16. Deshabilitar esta característica mejorará significativamente el número de cuadros por segundo en las tarjetas gráficas que no soporten filtrado antisotrópico.
    SET farclip "477.0"
    Esto concuerda con la barra de desplazamiento de FarClip del menú de opciones de vídeo. El rango es de 177 a 1177. Esta característica tiene un pequeño impacto sobre el rendimiento.
    SET pixelShaders "0"
    Activar pixelShaders permite que el mundo sea renderizado cada pocos pasos, potencialmente incrementa el rendimiento en tarjetas que soportan este tipo de opciones. Quizá quieras probar la velocidad del juego con esta opción tanto activada ("1") como desactivada ("0"). Está actualmente deshabilitada para todas las versiones Macintosh del juego.
    SET specular "0"
    Specular no se puede usar sin pixelShaders puesto a "1". Esta opción activa el realce de ciertas texturas del terreno, dando una brillante y reflectante visión a algunos objetos. Activar la especularidad ("1") introduce una ligera disminución del rendimiento.
    SET fullAlpha "0"
    Como muchas tarjetas no soportan esta opción, es recomendable que desactives esta función poniéndola a "0".
    SET MaxLights "1"
    Puede ser desde 1 a 4. Muchas tarjetas de vídeo pueden soportar múltiples luces bien, pero aquellas que no experimentarán un aumento de la velocidad reduciendo este número.
    SET gxVSync "0"
    Esto desactiva la sincronización vertical, que permitirá algún desgarrón vertical. Colocándolo a 1 disminuirá la velocidad del juego pero aumentara la calidad visual.
    SET frillDensity "1"
    Se puede configurar entre 1 (apagado) y 128. Más alto sea el número, mayor es el detalle del entorno (hierba, piedras, etc.) Un mayor detalle disminuye el rendimiento.

Otra manera de mejorar el rendimiento en un PC es ajustar la apertura AGP de la tarjeta de vídeo (si está disponible). La norma general es tener la apertura colocada al doble (x2) de la cantidad de memoria de vídeo de la tarjeta gráfica. Por lo que si tienes una tarjeta gráfica de 64MB, configura la apertura AGP a 128. Puedes ajustar esta cantidad más adelante si es necesario. Nota: Este es un paso más avanzado. Blizzard Entertainment no te puede proporcionar instrucciones para configurar la apertura AGP de una tarjeta gráfica. Contacta al fabricante de tu computadora o tarjeta gráfica para obtener más ayuda.

2. La experiencia de juego es lenta o entrecortada en las ciudades:  si estás experimentando un número de cuadros por segundo muy pobre cuando entra en grandes áreas de las ciudades, por favor intente cambiar el método de renderizado entre "OpenGL" y "Direct 3D".

  • Abre el directorio de World of Warcraft de tu disco duro.
  • Abre la carpeta WTF.
  • Abre el archivo Config.wtf usando un editor de texto, como Bloc de notas.
  • Elije SET gxApi "opengl" o SET gxApi "direct3d"
  • Guarda el fichero e inicia World of Warcraft: Wrath of the Lich King de nuevo.

3. Las texturas del juego brillan, parpadean o están deformadas:  algunas tarjetas gráficas pueden experimentar problemas mostrando ciertos efectos de texturas mientras está jugando. Algunos de estos problemas gráficos pueden incluir lo siguiente: brillos de colores a través de texturas, texturas que bloquean de un verde sólido o blanco, geometría que falta o texturas que no están.

Para resolver estos problemas, pon la specularity y pixelShaders a "0"

  • Abre el directorio de World of Warcraft de tu disco duro.
  • Abre la carpeta WTF.
  • Abre el archivo Config.wtf usando un editor de texto, como Bloc de notas.
  • Cambia SET specular "1" a SET specular "0".
  • Cambia SET pixelShaders "1" a SET pixelShaders "0".
  • Guarda el fichero e inicia World of Warcraft: Wrath of the Lich King de nuevo.

4. Las propiedades gráficas están corruptas o no se configuran adecuadamente:  puedes restaurar las opciones por defecto en cualquier momento borrando el archivo Config.wtf. Este archivo está situado en tu directorio de World of Warcraft dentro de la carpeta WTF. Un nuevo archivo Config.wtf se creará la próxima vez que ejecute el juego. Las configuraciones del juego que cambiaron durante la última sesión de juego no estarán completamente actualizadas hasta que no salga del juego.
