Problemas de vídeo (Mac)

La mayoría de los errores de vídeo tienen sus raíces en el software y la configuración. Si sufre un problema que no está en la siguiente lista, por favor contacte con el departamento de soporte técnico.

1. El juego es lento o funciona a saltos:  El mejor modo de mejorar el rendimiento de su juego es ajustando la configuración gráfica dentro del juego. Cambiar a una configuración menor puede incrementar la velocidad y suavidad del juego, al coste de calidad visual. Tal vez quieras experimentar con diferentes configuraciones para encontrar la mejor calidad visual con una velocidad aceptable.

Algunas configuraciones pueden ser ajustadas mediante el menú de Opciones de Vídeo, pulsando Esc dentro del juego. Las configuraciones también pueden ser ajustadas mediante el archivo "Config.wtf".

  • Abre la carpeta de World of Warcraft de tu Disco Duro.
  • Abre la carpeta WTF.
  • Abre el archivo "Config.wtf" con un editor de texto.

Estas configuraciones se crearon para dar a los usuarios avanzados la posibilidad de personalizar el rendimiento del juego. Para mejorar la velocidad de rendimiento rápidamente, prueba a añadir o editar las siguientes instrucciones.

    SET lod "1"
    Cambia el nivel de detalle del terreno (LOD) entre activado (1) o desactivado (0). Habilitar esta característica permite al juego dibujar menos detalles cerca del horizonte, incrementando el rendimiento.
    SET lodDist "100"
    Determina la distancia a la que el nivel de detalle del terreno (LOD) cambiará. 100 es óptimo, pero puedes querer reducir este número a 50 para mejor rendimiento. LodDist solo funciona cuando LOD es "1".
    SET doodadAnim "0"
    Cambiar esto a "1" activará las animaciones ambientales, tales como molinos de viento y pájaros. Algunas zonas del juego tienen muchos objetos animados, así que dejar esto en 0 puede incrementar el rendimiento significativamente en dichas zonas.
    SET mapShadows "0"
    Desactiva la visualización de las sombras del terreno ("1" las reactiva). Esto mejorará la velocidad del juego ligeramente.
    SET anisotropic "1"
    Puede cambiarse desde 1 (desactivado) a 16. Desactivar esta característica incrementará significativamente los fotogramas en tarjetas gráficas que no soportan filtrado anisotrópico.
    SET farclip "477.0"
    Esto coincide con la barra de desplazamiento de Clip lejano en las Opciones de Vídeo. El rango es de 177 hasta 777. Esta opción tiene poco impacto en el rendimiento.
    SET pixelShaders "0"
    Habilitar el sombreado de píxeles permite que mejorar el mundo en menos tiempo, incrementando potencialmente el rendimiento con tarjetas que lo soportan. Tal vez quieras probar la velocidad con este ajuste activado ("1") o desactivado ("0"). Esto está actualmente desactivado para las versiones Macintosh del juego.
    SET specular "0"
    La luz especular no puede ser usada si pixelShaders no es "1". Esta opción habilita el resaltado especular en ciertas texturas del terreno, dando una apariencia brillante y reflectiva a algunos objetos. Activar la especularidad ("1") impacta ligeramente en el rendimiento.
    SET fullAlpha "0"
    Ya que no muchas tarjetas soportan esta característica todavía, se recomienda que esta opción sea desactivada cambiando su valor a "0".
    SET MaxLights "1"
    Puede establecerse desde 1 a 4. Muchas tarjetas de vídeo manejan múltiples luces de manera adecuada, pero aquellas que no pueden disfrutarán de una velocidad notablemente mejor bajando este valor.
    SET gxVSync "0"
    Esto desactiva la sincronización vertical, lo cual permite un "desgarre" vertical. Estableciéndolo en 1 bajará la velocidad del juego, pero incrementará su calidad visual.
    SET frillDensity "1"
    Puede establecerse entre 1 (desactivado) y 128. Cuanto más alto sea este número, más detalles contendrá el terreno (hierba, piedras, etc.). A más detalle, menos rendimiento.

2. Las texturas del juego brillan, parpadean, o están distorsionadas:  Algunas tarjetas de vídeo pueden sufrir problemas visualizando ciertos efectos de texturizado mientras juegas. Problemas gráficos pueden incluir lo siguiente: colores cambiantes a través de texturas estáticas, texturas de color verde o blanco sólido y cuadradas, falta de geometría, o falta de texturas.

Para solucionar estos problemas, cambia "specularity" y "pixelShaders" a "0"

  • Abre la carpeta de World of Warcraft en el disco duro.
  • Abre la carpeta WTF.
  • Abre el archivo "Config.wtf" usando un editor de texto, como por ejemplo, el Bloc de notas.
  • Cambia SET specular "1" a SET specular "0".
  • Cambia SET pixelShaders "1" a SET pixelShaders "0".
  • Guarda el archivo y reinicia World of Warcraft.

3. Las opciones gráficas están dañadas o no debidamente estabilizadas:  Puedes restaurar la configuración del juego en cualquier momento borrando el archivo "Config.wtf". Este archivo se encuentra en su directorio de World of Warcraft, dentro de la carpeta WTF. Un nuevo archivo "Config.wtf" será creado la próxima vez que se ejecute el juego. Los ajustes que se hayan cambiado durante la última sesión de juego no serán totalmente actualizados hasta que salga del juego.
